About This Project
Technical details, model documentation, and dataset information

🏗️ Architecture

CardioRisk AI uses a multi-model ensemble approach where 6 independent Gradient Boosting Classifier (GBM) models are trained on a synthetic clinical dataset of 7,500 patients. Each model targets a different condition using an overlapping set of clinical features. A secondary Logistic Regression layer is trained on the same data for fast in-browser inference.

Python 3.11scikit-learnGBMLogistic RegressionMinMaxScalerHTML/JS

📊 Dataset

Trained on a synthetic dataset of 7,500 patients generated with clinically realistic correlations between features. CVD prevalence is 49.5% (balanced for training). Features include demographics, vitals, labs, comorbidities, and lifestyle scores. The dataset schema aligns with the Framingham Heart Study and CDC BRFSS health survey formats.

⚠️ Disclaimer

This tool is developed for a Hackathon and is intended for educational and research purposes only. It is NOT a medical device and should not be used for actual clinical diagnosis or treatment decisions. All predictions are probabilistic estimates based on synthetic training data. Always consult a qualified healthcare professional for medical advice.

🔗 Features Used

25 clinical features across demographics, vitals, labs, and lifestyle: Age, Sex, Marital Status, Race/Ethnicity, Income, Height, Weight, BMI, Systolic BP, Diastolic BP, Heart Rate, SpO₂, Respiratory Rate, HbA1c, PHQ-2 Score, Alcohol Score, and binary flags for Diabetes, Hypertension, Sleep Disorder, Stress/Anxiety, Obesity, and High Cholesterol.
